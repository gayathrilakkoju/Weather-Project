# Weather App

##  Description
This is a simple weather application where users can enter a city name and view current weather details.

## Features
- Search weather by city name  
- Shows temperature, humidity, and weather condition  
- Displays error message for invalid city  
- Simple and clean user interface  

## How It Works
The app gets weather data from an online service and shows it on the screen when the user enters a city name.

## Technologies
- HTML  
- CSS  
- JavaScript
